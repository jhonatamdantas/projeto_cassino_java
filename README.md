# Projeto Java

<p>O objetivo da criação foi aplicar os conceitos vistos na faculdade sobre programação orientada a objetos com Java</p>

## Descrição
<p>O projeto consiste em um jogo de cassino onde o jogador tem a chance de ganhar ou perder dinheiro.</p>

## Função de cada arquivo

### Main.java
<p>Classe principal do jogo.</p>

### Animacao.java
<p>Classe responsável pela animação do jogo.</p>


### PoolSimbolo.java
<p>Classe responsável por sortear os 3 símbolos que irão aparecer na animação.</p>

### Simbolo.java
<p>Classe que representa os símbolos que irão aparecer na animação.</p>
